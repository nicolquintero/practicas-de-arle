¿Qué es Java?

Java es un lenguaje de programación muy utilizado para crear aplicaciones de escritorio, aplicaciones web y aplicaciones móviles. Es un lenguaje seguro, robusto y multiplataforma.

Conceptos Básicos de Java

  • Variables y tipos de datos
  • Condicionales (if, switch)
  • Ciclos (for, while, do-while)
  • Métodos
  • Arrays y ArrayList

Ventajas de Java

Java permite escribir un programa una sola vez y ejecutarlo en diferentes sistemas operativos. Además, es uno de los lenguajes más solicitados en el mundo laboral.

Variables en Java

Una variable es un espacio en memoria donde se guarda un dato. En Java, cada variable debe tener un tipo de dato. 

int edad = 19;
double nota = 4.5;
String nombre = "Nicole";

Operadores en Java

Los operadores permiten realizar operaciones sobre variables y valores.

  • Aritméticos: + - * / %
  • Relacionales: == != > < >= <=
  • Lógicos: && || !
int a = 10;
int b = 5;
System.out.println(a + b); // suma
System.out.println(a > b); // comparación

Estructura if - else

La estructura if - else se usa para tomar decisiones en el programa según una condición. 

int edad = 20;

if (edad >= 18) {
    System.out.println("Mayor de edad");
} else {
    System.out.println("Menor de edad");
}

Operador Ternario

El operador ternario es una forma corta de escribir un if - else. 

int edad = 20;
String mensaje = (edad >= 18) ? "Mayor de edad" : "Menor de edad";
System.out.println(mensaje);
